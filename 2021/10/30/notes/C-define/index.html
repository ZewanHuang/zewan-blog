C语言宏定义用法

本文最后更新于：2021年10月30日 凌晨

最近在用 C 语言写编译作业时，想实现支持存储多类型的栈，发现宏定义是很好的办法，就自学了一下，顺便简练记录下C语言宏定义的基本用法。

#define 命令是 C 语言中的一个宏定义命令，它用来将一个标识符（宏名）定义为一个字符串，该标识符被称为宏名，被定义的字符串称为替换文本。程序编译之前，编译的时候所有的宏名都会被定义的字符串替换，这便是宏替换。

C 语言编译的第一步是预处理阶段，宏正是在该阶段发挥作用。预处理器在源代码编译前对其进行一些文本性质的操作，主要包括插入 #include 进来的文件内容、替换 #define 定义的符号等。文本性质的操作，指的是简单粗暴地插入文本或将文本替换为另一段文本，而不考虑语义。

#define 的用法包含定义符号和定义宏 macro，下面分别介绍。
